本工具旨在于使用柱状图动态表示算法，使得其得以可视化，目前已完成冒泡排序可视化与选择排序可视化。
## 使用语言
html；css；JavaScript
## 主要特点
将算法从代码可视化；
从代码可视化算法；
了解算法；
